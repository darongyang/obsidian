# Make LLM Inference Affordable to Everyone: Augmenting GPU Memory with NDP-DIMM

## Motivation

（1）NDP-DIMM是什么？

NDP-DIMM（近数据处理内存模块）在内存模块内部直接部署处理单元（如GEMV单元、激活单元等），支持在数据存储位置附近完成部分计算任务，从而减少数据在GPU与主机内存间的传输开销。

（2）文中两个相似性的观察是什么？

令牌间相似性（Token-wise Similarity）：相邻令牌的激活神经元分布高度相似（如90%以上），而距离较远的令牌相似性逐渐降低。这种特性源于上下文语义的连续性，例如相邻令牌通常表达相近含义。

层间相关性（Layer-wise Correlation）：相邻层的激活神经元存在强关联。例如，某一层的特定神经元激活时，下一层的对应神经元激活概率显著提升（如超过90%）。这种相关性可通过历史激活信息预测当前层神经元活动。

（3）带来的挑战是什么？

动态冷热神经元划分：由于激活稀疏性随输入动态变化，需实时调整神经元的冷热分类。传统MLP预测器开销大，需设计轻量级预测器。

多NDP-DIMM间的负载均衡：激活神经元的动态分布导致不同NDP-DIMM的计算负载不均衡（如1.2-2.5倍差异）。

## Key Idea

利用稀疏激活减少数据传输（offline）

利用token间相似性和层间相似性，调整冷热神经元划分

## Design and Solution

（1）文章的设计是什么？

*   离线划分，就是一个优化问题，和powerinfer一样
*   利用相邻token的神经元相似性预测活跃神经元，维护神经元状态表
*   利用神经元的相关性来预测相邻层的活跃神经元，维护神经元相关表
*   冷神经元在线重映射实现负载均衡
