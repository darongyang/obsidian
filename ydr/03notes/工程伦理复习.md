- 分析模板
	- 背景-风险-责任-防范
	- 背景-伦理-责任
- 工程风险
	- 来源与防范
		- 概念：概率×损失
		- 来源：技术因素、外部环境因素、人为因素
		- 安全等级划分
		- 防范：质量监测、事故预防、应急处置
	- 伦理评估
		- 原则：以人为本、预防为主、整体主义和制度约束
		- 途径：专家评估（成本-收益）、社会评估（民众利益）、公众参与
		- 主体：工程内部、工程外部
		- 程序：信息公开、确定利益相关者、按照民主原则
		- 效力考察原则：公平、和谐和战略
	- 伦理责任
		- 不等于法律责任，不等于职业责任
		- 工程共同体
		- 分类：职业伦理责任、社会伦理责任、环境伦理责任
- 工程价值利益与公正
	- 工程价值及特点
		- 工程价值的多元性：科学、政治、社会、文化和生态等
		- 继：工程的内在价值、“双刃剑”、应对工程批评
		- 工程价值的综合性：各方面的价值最大化
	- 工程服务对象和可及性
		- 目标人群：按购买力标准来排除
		- 可及与普惠：价格、群体的知识水平
		- 成本承担：邻避效应、社会成本
		- 利益攸关方：影响企业目标或被影响的人
		- 公平原则：
			- 利益补偿
				- 普遍约束力的分配与补偿原则
				- 社会评价、后果评估、利益相关者分析
			- 公众参与
- 工程师职业伦理
	- 地位：劳动者、边缘人
	- 表现：对职业、对人（上司，同行）
	- 阶段：工程决策（防范），工程实施（活动），工程应用（承担后果）
	- 职业伦理：
		- 三种责任：义务（主动）责任、过失（被动）责任和角色（本职）责任
		- 合理关照：能力越大，关照越大
		- 善举：义务责任
		- 组织责任，个人责任，多人责任与懈怠
		- 被动责任（追究，难以承担）、主动责任（最坏情况，知情同意）
		- 负责的障碍：私利、害怕、自欺、无知、团队
		- 国际责任：技术迁移、伦理关联、国际权利
	- 职业道德：一般又特殊、员工、企业
	- 必备技能：知识、创新、风险、机遇
- 信息与大数据的伦理问题
	- 数据本质：生产资料和资产
	- 伦理问题：隐私、收益、虚假、匿名、公平、产权
	- 伦理责任：科技人员，个人自由，技术保护，行业自律，社会责任
- 伦理决策和学术道德
	- 伦理决策
		- 道德直觉能力
		- 启发式方法：伦理分析记录 -> 遵循伦理观 -> 描述案例 -> 传统解决方案 -> 自身伦理知识/他人的意见 -> 多种系统分析方法 -> 得出伦理结论 -> 对未来的启发
	- 学术道德
